Getting Started with Markdown

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Created by John Gruber in 2004, Markdown is now one of the world's most popular markup languages.

Why Use Markdown?

Using Markdown is different than using a WYSIWYG editor. In an application like Microsoft Word, you click buttons to format words and phrases, and the changes are visible immediately. Markdown isn't like that.

  • It's easy to learn and use
  • Platform independent
  • Future-proof and portable

Basic Syntax

Here are some basic markdown elements:

  1. Headers using # symbols
  2. Emphasis with asterisks or underscores
  3. Lists with dashes or numbers
